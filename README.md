## Description

Model attributes serialize to hash checker.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'rubocop-serializer', require: false
```
Then include cop in your `.rubocop.yml` like this:
```YAML
require:
  - rubocop-serializer
```
